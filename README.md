# 🌍 地理学习 - 交互式教学项目

这是一个使用 Three.js 开发的交互式地理教学项目，通过 3D 可视化动画帮助学生更好地理解地理知识。

## 📚 项目内容

### 地理选修1：自然地理基础 (Geography Elective 1: Foundation of Physical Geography)

#### 1. 🌀 地球自转 (Earth Rotation)
- **文件位置**: `geography-elective-1-foundation-of-physical-geography/earth-rotation/index.html`
- **主要内容**:
  - 地球自转的定义与方向（自西向东）
  - 自转周期：太阳日（24小时）与恒星日（23小时56分4秒）
  - 角速度与线速度的概念
  - 不同纬度的线速度对比
  - 多视角观察（北极俯视、南极仰视、赤道侧视等）

#### 2. ☀️ 黄赤交角与回归线 (The Obliquity of the Ecliptic)
- **文件位置**: `geography-elective-1-foundation-of-physical-geography/obliquity-of-ecliptic/index.html`
- **主要内容**:
  - 黄赤交角的定义（约23.5°）
  - 地球公转与四季变化
  - 太阳直射点的移动规律
  - 北回归线与南回归线
  - 春分、夏至、秋分、冬至四个节气
  - 中国在地球上的位置标注

## 🚀 技术栈

- **Three.js**: 3D 图形渲染
- **HTML5/CSS3**: 页面结构与样式
- **原生 JavaScript**: 交互逻辑
- **响应式设计**: 支持移动端访问

## 💡 功能特点

- ✨ 真实地球纹理与云层效果
- 🎮 交互式控制面板（暂停、调速、视角切换）
- 📖 内置知识卡片，支持滑动浏览
- 🔄 动态演示地球自转与公转
- 📍 地理坐标与纬线标注
- 🌟 北极星与地轴指向可视化

## 📖 使用方法

1. 直接在浏览器中打开对应的 `index.html` 文件
2. 使用右侧控制面板调整动画速度
3. 点击不同视角按钮切换观察角度
4. 点击左下角 "?" 按钮查看详细知识点
